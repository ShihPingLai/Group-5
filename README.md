## **使用說明：**
### 水波槽模擬：<br />
需先將Ripple Tank Simulation資料夾中的檔案下載下來，分別有
1.  無雜訊
2.  0.1倍強度高斯白雜訊
3.  1倍強度高斯白雜訊
4.  2倍強度高斯白雜訊
5.  10倍強度高斯白雜訊<br />

(以 *Python 3* 執行檔案，即可看到模擬水波槽的動畫。)<br />

在大約加上2倍強度的高斯白雜訊時，水波槽的動畫會如同真實的水波一般，產生立體感。<br />
以應用來說，像是海面的影像，距離較遠的話僅能看出亮暗變化，若加上雜訊後使其產生隨機共振，可增強其對於海面的波浪辨識度。
